Piggy Map Rules

Rule 1 No Asking for Build

This is a custom Piggy roleplay/minigame map. Do not ask the host to give you build mode permissions or access to modify the map. The kitchen layout is already set!

Rule 2 No Begging to Start

Please be patient while waiting in the lobby. Do not spam the chat asking the host to type "/start". The cooking will begin as soon as everyone is in the server and ready.

Rule 3 Don't Be Inappropriate

Keep the chat friendly and respect the Roblox Terms of Service. Any toxic behavior, bullying, trolling, or inappropriate roleplay will get you kicked from the server.

How to Build This Map in Piggy

  • Open Piggy Build Mode.
  • Create a large room using Basic blocks.
  • Place counters around the edges.
  • Group Cyan Player Spawns together.
  • Place the Red Piggy Spawn far away.
  • Add stoves, fridges, and shelves from Decorations.
  • Place items (keys) and link them to doors.
  • Type /save [slot number] to save.
